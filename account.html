    •   Account Page

    There are four subpages in account page: Checking; Transaction; Summary; Encryption

    1. Checking subpage: manage all the checks that belong to you.
      • Get all owned check - double click the tag of "Checking" subpage or click "Renew Checks" button.
      • Open collector check table - click "Collector Checks" button.
      • Open your account - input your account password; click "Open" button.
      • Backup all account info with specified name to bin fold - open your account; input file name; click "Backup" button.
      • Find the privkey for your pubkey - open your account; input pubkey; click "Get Privkey" button.
      • Verify a check - input a check; click "Verify" button.
      • Save a saved account file into your node - open your account; input the saved account file name; click "Account" button.
      • Save a privkey into your node - open your account; input a privkey; click "Privkey" button.
      • Save a pubkey into your node - input a pubkey; click "Pubkey" button.
      • Save a check into your node - input a check; click "Check" button.
    2. Transaction subpage: setup your account
        3. 在2里输入代币代码；或输入“#get”和一个数量；或输入“#skip”和一个数字；或输入“#watch”；点击“查询交易”； 将在1里显示出相应代币代币的，指定数量的，不包括指定数字的前几个的，包括与观察性支票关联的全部交易。 没有任何输入； 点击“查询交易”；将在1里显示出所有与自己有关联的最近的十个交易。
      • Clean blocking nodes list - click "Clean Blocking" button.
      • Open connection nodes list - click "Connection List" button.
    3. Summary subpage: setup your local node
        4. 双击“账户总汇”标签栏；将显示该节点拥有的，可使用的，全部的代币总汇。
      • Open connection nodes list - click "Connection List" button.
    4. Encryption subpage: setup your local node
      • Blocking nodes setting
        • Add a node to blocking list - (1) input the node address tobe blocked; (2) input the seconds tobe blocked; click "Modify Blocking List" botton.
        • Delete a node from blocking list - (1) input "-" plus the node address tobe deleted from blocking list; click "Modify Blocking List" botton.
      • Connection nodes setting
        • Add a node to connection list - (3) input the node address tobe connected; click "Modify Connection List" botton.
        • Delete a node from connection list - (3) input "-" plus the node address tobe deleted from connection list; click "Modify Connection List" botton.
        信息加密过程如下： 在1里输入本节点拥有的支票代码；在2里输入想要加密的信息；在5里输入账户启用密码；点击“启用账户”；点击“加密信息”；将在4里显示出信息加密的结果-密语。 验证密语过程如下： 在1里输入本节点拥有的支票代码；在2里输入加密用的信息；在3里输入需要验证的密语；在5里输入账户启用密码；点击“启用账户”；点击“加密信息”；将在4里显示出密语验证的结果。
