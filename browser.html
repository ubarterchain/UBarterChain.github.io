    •   Browser Page

    There are two subpages in browser page: Search engine, Product list

    1. Search engine subpage: search information about transactions (TXs) in whole blockchain or local node's mempool.
      • Search blockchain
        • Search info of a TX - (1) input the TXID; click "Search" botton.
        • Search info of a block related to a TX - after find the info of a tx, click "Search" botton.
        • Search info of the newest block - input nothing or "-"; click "Search" botton.
        • Search info of a block - input block hash or specified block height; click "Search" botton.
        • Search info of next block - after find one block, click "Search" botton.
      • Search node's mempool
        • Search info of all TXs in local node's mempool - without any input, click "Search TX" botton.
        • Search info of all barterable TXs in local node's mempool - double click the tag of subpage "Search Engine".
        • Search info of all TXs with conditons - select or input conditions; click "Search TX" botton.
    2. Product list subpage: learn about the operation of barterone blockchain
      • Search for info of all listed products - double click the tag of "Product List".
      • Search for info of specified token - input the symbole of the token; click the "Find" button.
