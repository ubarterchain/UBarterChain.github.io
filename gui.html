   Install GUI

  Installation:  (reference video)

  1. Download Easychain.zip in page  Here。 (after open the page, click Download button on right hand side will begin the download.）
  2. Unzip Easychain.zip，then put the inside Easychain folder at the desktop of your computer
  3. Open Easychain folder，then click run-easycoin file, the GUI will begin to running.

 Note: after the GUI begin to running, if it's not syc data with the blockchain after sometime, you can refer following steps to improve it.

  Syc date with blockchain:  (reference video)

 To improve the syc data with blockchain, you can download some former blocks in file Easycoin to replace the Easycoin file in your computer as following steps:

  1. Download file Easycoin from  Here
  2. Unzip the Easycoin.zip file and put it at directory:\Users\xxxx\AppData\Roaming to replace the old Easycoin file. Here the xxxx part should be your company user name，such as yongr for me。

  if the Barteronecoin file in your computer is not located at above directory, your can find it in GUI's debug file. To find the directory is just find “Default data directory”, after that is the required directory.
