  Install GUI

Installation:  (reference video)

  • 1. Download Barteronechain.zip file from Here (click Download button on right hand side）.
  • 2. Unzip Barteronechain.zip and put it at the desktop of your computer
  • 3. Open Barteronechain folder，then click run-easycoin file, the GUI will begin to running.

    • Note: after the GUI begin to running, if it's not syc data with the blockchain after sometime, you can refer following steps to improve it.

    Syc date with blockchain:  (reference video)

  • 1. Download Barteronecoin.zip file from Here (click Download button on right hand side）.
  • 2. Unzip the Barteronecoin.zip file and put it at directory:\Users\xxxx\AppData\Roaming to replace the old Easycoin file. Here the xxxx part should be your company user name，such as yongr for me.

    • Note: if the Barteronecoin file in your computer is not located at above directory, your can find it in GUI's debug file. To find the directory is just find the “Default data directory”, follow it is the required directory.  (reference picture)
